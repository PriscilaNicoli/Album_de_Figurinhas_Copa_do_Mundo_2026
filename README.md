# Álbum de Figurinhas da Copa do Mundo 2026

Sistema desenvolvido em Java para gerenciar a coleção de figurinhas de um álbum físico da Copa do Mundo 2026.

## Objetivo

O objetivo do sistema é auxiliar um colecionador no controle das figurinhas do álbum, permitindo registrar figurinhas coladas, figurinhas repetidas, consultar figurinhas faltantes e visualizar informações por seleção e por tipo.

## Funcionalidades

- Registrar figura colada no álbum.
- Registrar figura repetida.
- Consultar total de figuras coladas.
- Consultar total de figuras repetidas.
- Listar figuras faltantes.
- Listar figuras faltantes por seleção.
- Listar figuras por tipo.
- Salvar os dados da coleção do usuário em arquivo.

## Conceitos de POO utilizados

O projeto utiliza conceitos de Programação Orientada a Objetos, como:

- Classes e objetos.
- Encapsulamento.
- Herança.
- Polimorfismo.
- Classe abstrata.
- Enumeração.
- Separação de responsabilidades em pacotes.
